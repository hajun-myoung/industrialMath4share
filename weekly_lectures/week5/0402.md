## CSS

- Style Prioirity
  - `div ~ div` 와 `.internal-css` 의 차이: 클래스 지정이 더 구체적이므로 우선적용

- `~` tilda selector
  - 중간에 끊긴 스타일

    ```html
    <div>div1</div>
    <div>div2</div>
    <h3>header3</h3>
    <div>div3</div>
    <div>div4</div>
    ```

    에서, `div ~ div` 를 적용하면, 두 번째와 네 번째만 선택됨

- `+` plus selector
  - 연속된 스타일

    ```html
    <div>div1</div>
    <div>div2</div>
    <div>div3</div>
    ```

    에 대해서

    ```css
    div + div {
      background-color: crimson;
    }
    ```

    을 적용 시
